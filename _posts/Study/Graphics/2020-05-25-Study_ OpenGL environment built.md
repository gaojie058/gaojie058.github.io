# Software
- [cmake](https://en.wikipedia.org/wiki/CMake)
   - a cross-platform free and open-source software tool
   - managing the build process of software
   - using a compiler-independent method
   - It is used in conjunction with native build environments such as Make, Qt Creator, Ninja, Apple's Xcode, and Microsoft Visual Studio
   - It has minimal dependencies, requiring only a C++ compiler on its own build system.
- [homebrew](https://en.wikipedia.org/wiki/Homebrew_(package_manager))
   - a free and open-source software [package](https://en.wikipedia.org/wiki/Package_(macOS)) management system
   - simplifies the installation of software on Apple's macOS operating system and Linux   

# For MAC
- install cmake `brew install cmake`
- `brew list` list all the packages installed using brew
- install [glew](https://en.wikipedia.org/wiki/OpenGL_Extension_Wrangler_Library) `brew install glew`
